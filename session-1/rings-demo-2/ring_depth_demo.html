Depth ≠ power, unless there's a nonlinearity

Claim: five stacked linear layers collapse mathematically into a single linear map, so a 5-layer linear network is no stronger than 1 layer. Both are restricted to a straight-line decision boundary, so both fail a task that needs a curve — a "ring" (two concentric classes). Insert a ReLU between those same five layers and the network can finally bend the boundary and solve it. This page trains all three on identical data and proves the collapse numerically.
